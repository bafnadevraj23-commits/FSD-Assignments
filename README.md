## Getting Started

To run any of the assignments locally, generally you'll follow these steps:
1. Navigate into the desired assignment directory.
2. Check the specific assignment's `README.md` for dedicated instructions.
3. If it is a frontend-only project (like Assignment 1-3), you can simply open the `index.html` file in your browser.
4. If it's a backend or Full Stack project (like Assignment 5 & 6), you may need to:
   - Run `npm install` in the backend and/or frontend folders
   - Setup Environment Variables (`.env`)
   - Start the server using `npm start` or `node server.js`
