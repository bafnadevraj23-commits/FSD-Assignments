# Assignment 4 - Student Management System

University Student Management System with CRUD operations.

## Technologies
- Node.js
- Express.js
- EJS (Templating)
- MongoDB (Mongoose)
- In-memory fallback

## Features
- CRUD operations for student records
- EJS templating for views
- MongoDB data persistence
- Add, view, edit, delete students

## Run
```bash
npm install
npm start
```
