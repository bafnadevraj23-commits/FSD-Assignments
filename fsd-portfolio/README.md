# FSD Portfolio

Professional portfolio website for Devraj Bafna.

## Technologies
- HTML5
- CSS3 (Advanced)
- JavaScript (Vanilla)
- Google Fonts (Inter, Outfit)
- Font Awesome

## Features
- Hero section with glitch text effect
- About/Summary with glass-morphism styling
- Skills grid (Backend, DevOps, Databases, ML, Tools)
- Featured projects with demo links
- Education and achievements timeline
- Contact section
- Custom cursor glow effect
- Scroll reveal animations

## Run
Open `index.html` in any browser.
